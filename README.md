Project 1 — Water Alarm Circuit

Overview
A water detection alarm circuit designed to alert users when water is detected on a surface. This was my first complete PCB design project, built entirely from scratch using KiCad.

How It Works
When water bridges the detection probes, it completes the circuit and triggers an alarm output. The circuit uses basic analog components to detect the low resistance path created by water conductivity.

What I Did
Drew complete schematic from scratch in KiCad
Selected and assigned correct through-hole footprints
Designed PCB layout with proper component placement
Ran Electrical Rules Check achieving zero errors
Generated 3D view for visual verification
